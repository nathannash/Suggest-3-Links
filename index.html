Exercise 1 - Web Forms.

Have your Web site provide a list of five interests (e.g., sports, music, art, dance, history, science, current affairs, etc.). Then ask users to select one or more interests from that list. As a result of that selection, have your program display three suggested Web site links per interest expressed.
